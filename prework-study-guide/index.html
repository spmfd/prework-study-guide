HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • !DOCTYPE - This declaration tells the browser what document type it should expect when it is opened
  • The title element defines the title for a webpage. This element is required and is extremely important for Search Engines
  • The base element defines the base URL for a webpage
  • The link element connects an external resource to the HTML document
  • The meta element defines metadata such as the character set, description, keywords, author and viewpoint
  • The charset element sets our character coding
  • The http-equiz element sets an HTTP header for content
  • the viewport element sets the default view for our webpage

Git

  • pwd - displays the current working directory
  • ls - displays the list of files & folders in the current directory
  • cd - navigates between the directory in the path
  • cd .. - navigates backwards one level at a time, through the path of your directories
  • mkdir - creates a new directory
  • touch - creates a file inside a directory
  • git status - checks which branch we are currently in
  • git checkout -b branch-name = creates a new branch and switches to it
  • code . - opens up VS code for branch we are in
